## 📋 Sobre o Projeto

API REST desenvolvida para um **Agregador de Investimentos**, permitindo o gerenciamento completo de usuários. O projeto foi construído seguindo boas práticas de desenvolvimento, com foco em simplicidade e eficiência.

A aplicação realiza operações **CRUD** (Create, Read, Update, Delete) em um banco de dados MySQL, utilizando o Spring Boot para expor endpoints REST.

---

## 🚀 Funcionalidades

- ✅ **Criar usuário** - Cadastro de novos usuários
- ✅ **Listar usuários** - Consulta de todos os usuários cadastrados
- ✅ **Buscar por ID** - Consulta de usuário específico
- ✅ **Atualizar usuário** - Edição de dados cadastrais
- ✅ **Deletar usuário** - Remoção de usuários do sistema

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Função |
|------------|--------|
| **Java 21** | Linguagem principal |
| **Spring Boot 3** | Framework para criação da API |
| **Spring Data JPA** | Mapeamento objeto-relacional |
| **Hibernate** | ORM para persistência |
| **MySQL** | Banco de dados relacional |
| **Maven** | Gerenciamento de dependências |

## 📌 Conceitos Aplicados

- Injeção de Dependência e Inversão de Controle (IoC)
- Arquitetura em camadas: Controller, Service, Repository
- Mapeamento de entidades com JPA
- Geração automática de UUID
- Padrão DTO para requisições
